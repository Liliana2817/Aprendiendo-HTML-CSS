¿QUÉ ES HTML?


Es un lenguaje de marcas de tipo descriptivo que son siglas de Hypertext Markup Language(HTML) diseñado para la creación de páginas web, el cual describe las diferentes partes en la que se estructura el documento, pero sin especificar cómo se presenta. Además, es un lenguaje versátil, fácil de adaptar y que permite una amplia variedad de usos, es decir, es flexible. La información está separada.

¿De qué está formado HTML?

HTML (Hypertext Markup Language) está formado por una serie de elementos, cada uno representando una parte específica del contenido de una página web. Cada elemento se define mediante etiquetas, y estas etiquetas se utilizan para estructurar y describir la información que se presenta en una página web. Aquí hay algunos conceptos clave:

      Etiquetas

    Las etiquetas son los bloques fundamentales de HTML y se utilizan para marcar y estructurar el contenido. Cada etiqueta tiene una apertura<etiqueta> y un cierre </etiqueta> y el contenido va entre ellas. Ejemplos de etiquetas incluyen <p> para párrafos, <h1> para encabezados, <a> para enlaces, entre otros.
      Atributos

    Las etiquetas pueden tener atributos que proporcionan información adicional sobre el elemento. Los atributos se incluyen en la etiqueta de apertura y tienen la forma nombre="valor". Por ejemplo, el atributo "href" en la etiqueta <a> se utiliza para especificar la URL de destino de un enlace.
      Elementos anidados:

    Puedes anidar elementos colocando un elemento dentro de otro. Por ejemplo, un párrafo (<p>) puede contener un enlace (<a>).
      Atributos globales:

    Algunos atributos son globales y se pueden utilizar en cualquier elemento. Por ejemplo, el atributo "class" se usa para aplicar estilos mediante CSS, y el atributo "id" proporciona una identificación única para el elemento.

Estructura básica de HTML

La estructura básica de HTML consta de los siguientes elementos:




Estructura HTML Elementos que forman la estructura de HTML
<!DOCTYPE html>
<html>
<head>
<title>Título de la página</title>
</head>
<body>
<h1>Este es un encabezado</h1>
<p>Este es un párrafo.</p>
</body>
</html>

  • <!DOCTYPE html>:
    Este elemento define la versión de HTML que se está utilizando en el documento. Este varia según la versión que se use, en este caso, HTML5.

  • <html>:
    Este es el elemento raíz que envuelve todo el documento HTML.

  • <head>:
    La información y configuraciones contenidas en <head>son esenciales para la correcta interpretación y presentación del contenido de la página, pero no son directamente visibles para los usuarios en el navegador.

  • <title>:
    Define el título del documento, que se muestra en la barra de título del navegador.

  • <meta>:
    Se utiliza para proporcionar información adicional sobre el documento HTML.

  • <link>:
    Comúnmente se utiliza para enlazar hojas de estilo CSS, íconos, fuentes, y muchos recursos más, con el documento HTML.

  • <style>:
    Se utiliza para escribir estilos CSS dentro del documento HTML.

  • <script>:
    Se utiliza para incrustar scripts, generalmente scripts de JavaScript, dentro del documento HTML, puede estar ubicado en <head> o en <body>

  • <body>:
    La etiqueta <body> contiene el contenido principal visible de la página web, incluye elementos como encabezados, párrafos, imágenes, enlaces, formularios, etc.







Etiquetas semánticas

Las etiquetas semánticas en HTML son aquellas que proporcionan información sobre el tipo de contenido que contienen, estos elementos semánticos permiten una mejor estructuración del contenido HTML, facilitan la accesibilidad para lectores de pantalla y motores de búsqueda, y mejoran la comprensión del propósito de cada sección del documento. Ejemplos de etiquetas semánticas en HTML:

Etiquetas Semanticas Definición
<header> Define la cabecera de una sección o de toda la página. También puede contener lncabezados, logotipos, enlaces de navegación, etc.
<nav> Representa la sección de navegación de la página. Contiene enlaces o elementos relacionados con la navegación.
<aside> Representa contenido relacionado pero secundario, como barras laterales o anuncios. Se utiliza típicamente dentro de un elemento <article> o <section>.
<main> Define el contenido principal de la página. Debe contener el contenido central y esencial de la página, excluyendo encabezados, pies de página y barras laterales.
.
<article> Representa un artículo o una pieza independiente de contenido que tiene sentido por sí misma. Puede ser un post de blog, un artículo de noticias, un comentario, etc.
<section> Define una sección temática o agrupa contenido relacionado. Puede contener otros elementos semánticos o contenido diverso.
<footer> Define el pie de página de una sección o de toda la página. Contiene información como derechos de autor, enlaces de contacto, etc.

    Atributos HTML

    Los atributos contribuyen a la claridad y la funcionalidad del código HTML. Cada elemento tiene sus propios atributos, a continuación, te muestro algunos atributos:

    ATRIBUTOS HTML Descripción Ejemplo
    <class> Este atributo especifica una o más clases para un elemento HTML. <p class="ejemplo">Este es un párrafo de ejemplo.</p>
    <id> El atributo id se utiliza para asignar un identificador único a un elemento HTML. <div id="ejemplo">Un id de ejemplo</div>
    <style> El atributo style se utiliza para aplicar estilos en línea a un elemento HTML. <p style="color: blue; font-size: 16px;">Este párrafo tiene estilos en línea.</p>
    <lang> El atributo lang se utiliza para especificar el idioma principal del contenido del elemento. Se utiliza para ayudar a los motores de búsqueda y navegadores. <html lang="es">
    </html>
    <src> El atributo src se utiliza para especificar la fuente o la ubicación del recurso.Se utiliza habitualmente en las etiquetas <img>, <script> y <iframe>. <img src="Ejemplo_Imagen.jpg">
    <href> El atributo href se utiliza principalmente en enlaces (<a>) para especificar la URL de destino. Se utiliza en la etiqueta <a>. <a href="https://www.ejemplo.es">Dale click</a>
