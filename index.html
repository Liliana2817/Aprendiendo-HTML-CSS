ELIGE APRENDER HTML O CSS

HTML es el lenguaje utilizado para crear páginas web. Con él, estructuramos el contenido, como texto e imágenes, mediante etiquetas.

CSS es el lenguaje que da estilo a las páginas web. Controla la apariencia, colores y diseño, permitiendo que el contenido HTML se vea atractivo y coherente.

¿Cómo Accedo a los Contenidos?

Bienvenido a nuestra página web. A continuación, te proporcionamos una guía rápida sobre cómo utilizarla:

  • Navegación: Utiliza el menú de navegación de la izquierda para explorar las diferentes secciones de la página web.
  • Contenidos HTML: Para ir a la página de HTML puedes acceder desde el botón HTML de arriba o desde el menú de navegación de la izquierda.
  • Contenidos CSS: Para ir a la página de CSS puedes acceder desde el botón CSS de arriba o desde el menú de navegación de la izquierda.
  • Formulario de autoevaluación: Pon a prueba tus conocimientos realizando un test al que puedes acceder desde el menú de navegación.
  • Formulario de contacto: Si tienes alguna pregunta puedes ponerte en contacto desde el botón de contacto del footer (abajo de la página).
  • Preguntas frecuentes: Antes de ponerte en contacto con nosotros, puedes acceder a esta sección desde el menú de navegación y el footer, para ver si encuentras tu duda en alguna pregunta.
  • Practica tus conocimientos: Desde el menú de navegación puedes acceder a una página que te permitirá escribir en HTML y CSS, así como ver el resultado en tiempo real.

Si tienes alguna pregunta o problema, no dudes en ponerte en contacto con nuestro equipo de soporte
(puedes encontrar el botón de contacto más abajo).
